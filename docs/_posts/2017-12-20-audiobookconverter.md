### Installation Instructions
Just run these commands:  

`git clone https://github.com/ahahn94/AudioBookConverter.git`  
`cd AudioBookConverter`  
`./build.sh`  
`./install.sh`  

### Remove Instructions
`./remove.sh`

### Generate Java Documentation
`./generateDoc.sh`
